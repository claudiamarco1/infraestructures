# MacOS
### Check Python Version
python3 --version</br>
if necessary brew install python3

### Create Virtual environment in your Project folder
python3 -m venv venv

### Activate envionment
source venv/bin/activate

### Install dependencies
pip install -r requirements.txt

### Run Python main
python3 main.py

### Deactivate environment
deactivate
---------------------------------
