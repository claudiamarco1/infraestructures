## PROYECTO
*Crear una Máquina Virtual usando VirtualBox y/o UTM con alguna versión de Ubuntu/Debian.*

Desarrollar un código Python sencillo con el que Extraer, Transformar y Cargar datos:

La Extracción se hará usando la siguiente WEB API: https://randomuser.me/api/
La Transformación consistirá en la confección de algunas estadísticas usando datos de los usuarios descargados.
La Carga será la escritura de las estadísticas y, si se quiere, de los usuarios en un fichero, con formato o sin, a vuestra discreción.
Instalar toda la paquetería necesaria en la máquina virtual para poder ejecutar el código Python.

ENTREGA
Se deberá entregar la presentación en formato PDF, donde se detalle el proceso de la creación de la máquina virtual con algunas capturas de pantalla y explicaciones. También se deberá razonar el motivo de las librerías instaladas y una breve explicación del código desarrollado y su función.

Documento PDF (máximo 20 páginas), solo 1 entrega por grupo.

Nombre PDF: grXX_vbox.pdf (XX se corresponde al número de grupo)

Tiempo de presentación: 15 minutos como máximo

OBSERVACIONES
Trabajo Grupal
Código desarrollado usando Python
El código debe tener algún tipo de estrategia ETL por simple que sea:
descarga de datos de la API recomendada
transformar algo esos datos (estadísticas, gráficos, etc...)
cargar esos datos en algún tipo de storage persistente
Para los que uséis Apple M1/M2/M3 es más sencillo instalar UTM como hypervisor y Debian 12 como OS invitado.
